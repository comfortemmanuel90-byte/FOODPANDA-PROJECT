## 🍔 Food Panda Data Analysis Project ##

 ## 📌 Project Overview ##
This project analyzes Food Panda data to uncover insights into customer behavior, restaurant performance, and delivery trends. The analysis is visualized using Power BI to support data-driven decision-making.

## 🎯 Objectives ##
Understand customer ordering patterns
Identify top-performing restaurants and cuisines
Analyze delivery performance
Provide actionable business insights

## 🛠️ Tools & Technologies##
Power BI (Data visualization & dashboard creation)
Excel / CSV (Data source)

## 📊 Dashboard Features ##
Sales and revenue overview
Order trends by time and location
Top restaurants and cuisines
Delivery performance metrics
Interactive filters and slicers

## 📈 Key Insights ##
Peak order times and customer behavior trends
High-performing cuisines and vendors
Areas with delivery delays
Customer contribution to revenue

## 🚀 How to Use ##
Download the .pbix file
Open in Power BI Desktop
Explore the interactive dashboard
