## Food Panda Data Analysis Project ##

## Project Overview ##
This project focuses on analyzing Food Panda data to uncover insights into customer behavior, order patterns, and business performance. The goal is to transform raw data into meaningful insights that can support data-driven decision-making and improve operational efficiency.

## Objective ##
* To analyze customer ordering trends and behaviors
* To identify key factors influencing sales and performance
* To provide actionable insights for improving customer satisfaction and revenue

## Scope ##
This analysis covers key aspects of Food Panda operations, including:
* Customer orders
* Delivery performance
* Restaurant activity
* Revenue trends
*  Analysis Covers
* Order frequency and distribution
* Peak order times and trends
8 Customer purchasing behavior
* Revenue performance
* Delivery efficiency

## Methodology ##
* Data cleaning and preprocessing
* Handling missing values and inconsistencies
* Data transformation for analysis
* Visualization and dashboard creation
* Insight generation and interpretation

## Data Source ##
The dataset used for this project was obtained from a Food Panda-related data repository (public dataset or provided source). It includes information on orders, customers, restaurants, and deliveries.

## Tools ##
Power BI
Power BI was the primary tool used for this project to perform data analysis and visualization. It played a key role in transforming raw data into interactive and insightful dashboards.
Key uses of Power BI in this project include:
 * Data Cleaning & Transformation: Used Power Query Editor to handle missing values, format data types, and prepare the dataset for analysis
 * Data Modeling: Created relationships between tables to ensure accurate analysis
 * DAX Calculations: Developed measures and calculated columns for KPIs such as total revenue, average order value, and order count
 * Data Visualization: Designed interactive dashboards using charts, cards, slicers, and filters
 * Insight Generation: Enabled easy identification of trends, patterns, and performance metrics
   
## Exploratory Data Analysis (EDA) ##
During EDA, the following steps were performed:
* Examined data structure and summary statistics
* Identified missing values and outliers
* Explored relationships between variables
* Visualized trends using charts and graphs

## Key Questions to Address ##
* What are the peak ordering times?
* Which locations or restaurants generate the most revenue?
* What is the average order value?
*  does delivery time affect customer satisfaction?
* What patterns exist in customer behavior?

## Results ##
* Identified peak hours for customer orders
* Highlighted top-performing restaurants and locations
* Revealed trends in customer purchasing behavior
* Detected inefficiencies in delivery times
* a clear view of revenue distribution

## Recommendations ##
Optimize delivery operations during peak hours
Focus marketing efforts on high-performing locations
Improve delivery time to enhance customer satisfaction
Introduce promotions during low-demand periods
Leverage customer behavior insights for targeted campaigns
